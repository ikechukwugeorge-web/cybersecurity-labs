# sqlmap

## Overview

This exercise focused on using sqlmap to automate SQL Injection testing against the DVWA application.

## What the Screenshots Show

The screenshots demonstrate database enumeration using sqlmap. I identified the available databases, inspected tables within the DVWA database, and viewed data stored in the users table, including password hashes returned during the assessment.

## What I Learned

This exercise showed me how sqlmap can automate many of the manual SQL Injection techniques I previously performed. I learned how to enumerate databases, identify tables, inspect database contents, and understand how quickly an attacker can gather information from a vulnerable application.

Working through this lab also helped me appreciate the importance of secure coding practices, parameterized SQL queries, proper access controls, and regular security testing to reduce the risk of SQL Injection vulnerabilities.
